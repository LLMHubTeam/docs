# Completion API

Use the Completion API to generate text given a prompt.

<Endpoint
    method="POST"
    url="https://openrouter.ai/api/v1/completions"
    auth="Bearer API Key"
/>

## Request

<RequestExample
    url="https://openrouter.ai/api/v1/completions"
    method="POST"
    auth="Bearer API Key"
    headers={{
        "Content-Type": "application/json"
    }}
    body={{
        "model": "openai/gpt-3.5-turbo-instruct",
        "prompt": "Write a haiku about recursion in programming.",
        "max_tokens": 100
    }}
/>

### Body Parameters

| Name        | Type     | Required | Description |
|-------------|----------|----------|-------------|
| `model`     | string   | Yes      | The model to use for text generation. |
| `prompt`    | string   | Yes      | The input text prompt. |
| `max_tokens`| integer  | No       | The maximum number of tokens to generate. |

## Response

```json
{
  "id": "cmpl-123",
  "object": "text_completion",
  "created": 1677858242,
  "model": "openai/gpt-3.5-turbo-instruct",
  "choices": [
    {
      "text": "Code flows like water\nFunctions call themselves anew\nInfinite mirrors",
      "index": 0,
      "finish_reason": "stop"
    }
  ],
  "usage": {
    "prompt_tokens": 10,
    "completion_tokens": 20,
    "total_tokens": 30
  }
}
